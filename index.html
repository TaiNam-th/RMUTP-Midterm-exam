เรียนรู้ Numbers ใน Python

เริ่มต้นเรียนรู้การใช้งานตัวเลขในภาษา Python แบบง่ายๆ สำหรับผู้เริ่มต้น

เริ่มเรียนเลย

เนื้อหาการเรียน

ทำความรู้จักกับตัวเลขใน Python

  • ภาพรวมของตัวเลขใน Python
  • ประเภทของตัวเลข
  • การแปลงชนิดข้อมูลตัวเลข
เริ่มเรียน

ตัวอย่างโค้ด

                        
                            # ตัวอย่างการใช้งานตัวเลขใน Python
                            x = 10          # Integer
                            y = 3.14        # Floating point
                            z = 2 + 3j      # Complex number

                            # การคำนวณพื้นฐาน
                            print(f"x + y = {x + y}")
                            print(f"x * y = {x * y}")
                            print(f"y / x = {y / x}")
