บทเรียน Numbers in Python

เรียนรู้เกี่ยวกับการใช้งานตัวเลขในภาษา Python ตั้งแต่พื้นฐานไปจนถึงการประยุกต์ใช้งาน

สารบัญ

  • ภาพรวมของตัวเลขใน Python
  • ประเภทของตัวเลข
    • • Integer (จำนวนเต็ม)
    • • Floating Point (จำนวนทศนิยม)
    • • Complex Numbers (จำนวนเชิงซ้อน)
  • การแปลงชนิดข้อมูลตัวเลข

ภาพรวมของตัวเลขใน Python

Python มีตัวเลข 3 ชนิดพื้นฐานที่ใช้งานบ่อย:

  • Integer (int) - จำนวนเต็ม เช่น -5, 0, 42
  • Floating Point (float) - จำนวนทศนิยม เช่น 3.14, -0.001, 2.0
  • Complex Numbers (complex) - จำนวนเชิงซ้อน เช่น 2+3j, -1-2j

ข้อดีของระบบตัวเลขใน Python:

  • ไม่จำกัดขนาดของจำนวนเต็ม
  • รองรับการคำนวณทางวิทยาศาสตร์
  • มีฟังก์ชันคณิตศาสตร์ครบถ้วน

ประเภทของตัวเลข

1. Integer (จำนวนเต็ม)

จำนวนเต็มที่ไม่มีทศนิยม สามารถเป็นได้ทั้งค่าบวก ค่าลบ และศูนย์ 

                    
                        # ตัวอย่างการใช้งาน Integer
                        x = 10          # ค่าบวก
                        y = -5          # ค่าลบ
                        z = 0           # ศูนย์
                        big_num = 1_000_000  # ใช้ _ เพื่อให้อ่านง่าย

                        print(f"x = {x}, type: {type(x)}")
                        print(f"big_num = {big_num}")

2. Floating Point (จำนวนทศนิยม)

จำนวนที่มีจุดทศนิยม สามารถแสดงในรูปแบบปกติหรือสัญกรณ์วิทยาศาสตร์ 

                    
                        # ตัวอย่างการใช้งาน Floating Point
                        pi = 3.14159
                        price = 45.50
                        tiny = 1e-10    # 0.0000000001
                        huge = 1e8      # 100000000.0

                        print(f"pi = {pi}")
                        print(f"tiny in scientific notation = {tiny:e}")

3. Complex Numbers (จำนวนเชิงซ้อน)

จำนวนเชิงซ้อนประกอบด้วยส่วนจริงและส่วนจินตภาพ ใช้ j แทนหน่วยจินตภาพ 

                    
                        # ตัวอย่างการใช้งาน Complex Numbers
                        c1 = 2 + 3j
                        c2 = complex(1, 2)  # สร้างจาก constructor

                        print(f"c1 = {c1}")
                        print(f"Real part: {c1.real}")
                        print(f"Imaginary part: {c1.imag}")

การแปลงชนิดข้อมูลตัวเลข

Python มีฟังก์ชันสำหรับแปลงชนิดข้อมูลระหว่างตัวเลขประเภทต่างๆ 

                
                    # การแปลงเป็น Integer
                    print(f"int(3.75) = {int(3.75)}")        # ปัดเศษทิ้ง
                    print(f"int('100') = {int('100')}")      # แปลงจาก string
                    print(f"int(-3.75) = {int(-3.75)}")      # ปัดเศษทิ้ง

                    # การแปลงเป็น Float
                    print(f"float(25) = {float(25)}")        # แปลงจาก integer
                    print(f"float('3.14') = {float('3.14')}") # แปลงจาก string

                    # การแปลงเป็น Complex
                    print(f"complex(3, 4) = {complex(3, 4)}")

ข้อควรระวัง:

  • การแปลงเป็น integer จะปัดเศษทศนิยมทิ้งเสมอ
  • ไม่สามารถแปลงจำนวนเชิงซ้อนเป็น integer หรือ float ได้โดยตรง
  • ควรใช้ try-except เพื่อจัดการข้อผิดพลาด

ตัวอย่างการจัดการข้อผิดพลาด:

                    
                        def safe_convert_to_int(value):
                        try:
                            return int(value)
                        except ValueError:
                            print(f"ไม่สามารถแปลง '{value}' เป็น integer ได้")
                        return None

                        # ทดสอบ
                        print(safe_convert_to_int("123"))    # 123
                        print(safe_convert_to_int("12.34"))  # แสดงข้อความผิดพลาด
                        print(safe_convert_to_int("abc"))    # แสดงข้อความผิดพลาด
