แบบฝึกหัด

ฝึกฝนการใช้งานตัวเลขใน Python ผ่านโจทย์ปัญหาต่างๆ

การคำนวณพื้นฐาน

1. การคำนวณพื้นที่สี่เหลี่ยม

จงเขียนโปรแกรมคำนวณพื้นที่สี่เหลี่ยมผืนผ้า โดย:

  • รับค่าความกว้างและความยาวจากผู้ใช้
  • คำนวณพื้นที่
  • แสดงผลลัพธ์พร้อมหน่วย
                                
                                    # รับค่าจากผู้ใช้
                                    width = float(input("ความกว้าง (เมตร): "))
                                    height = float(input("ความยาว (เมตร): "))

                                    # คำนวณพื้นที่
                                    area = width * height

                                    # แสดงผลลัพธ์
                                    print(f"พื้นที่สี่เหลี่ยมผืนผ้า = {area:.2f} ตารางเมตร")

2. การแปลงอุณหภูมิ

จงเขียนโปรแกรมแปลงอุณหภูมิจากเซลเซียสเป็นฟาเรนไฮต์ โดย:

  • รับค่าอุณหภูมิในหน่วยเซลเซียส
  • ใช้สูตร (C × 9/5) + 32
  • แสดงผลลัพธ์ทั้งสองหน่วย
                                
                                    # รับค่าอุณหภูมิ
                                    celsius = float(input("อุณหภูมิ (°C): "))

                                    # คำนวณ
                                    fahrenheit = (celsius * 9/5) + 32

                                    # แสดงผล
                                    print(f"{celsius:.1f}°C = {fahrenheit:.1f}°F")

การแปลงชนิดข้อมูล

3. แปลงข้อความเป็นตัวเลข

จงเขียนฟังก์ชันที่:

  • รับข้อความที่เป็นตัวเลขจากผู้ใช้
  • แปลงเป็นจำนวนเต็มหรือทศนิยม
  • จัดการข้อผิดพลาดที่อาจเกิดขึ้น
                                
                                    def convert_to_number(text):
                                    try:
                                    # ลองแปลงเป็นจำนวนเต็ม
                                        return int(text)
                                    except ValueError:
                                    try:
                                    # ถ้าแปลงเป็นจำนวนเต็มไม่ได้ ลองแปลงเป็นทศนิยม
                                        return float(text)
                                    except ValueError:
                                    # ถ้าแปลงไม่ได้ทั้งคู่
                                        return None

                                    # ทดสอบฟังก์ชัน
                                    test_values = ["123", "45.67", "abc", "12.0"]
                                    for value in test_values:
                                        result = convert_to_number(value)
                                        print(f"'{value}' -> {result} ({type(result) if result is not None else 'Invalid'})")
